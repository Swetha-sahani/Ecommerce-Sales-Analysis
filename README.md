# E-Commerce Sales Analysis Using Python

## Project Overview

This project analyzes e-commerce sales data to identify profitable categories, regional performance, and loss-making products.

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Google Colab

## Dataset

Sample Superstore Dataset (9,994 records)

## Key Findings

- Technology generated the highest sales and profit.
- Furniture showed high sales but low profitability.
- West region contributed the highest revenue.
- California was the top-performing state.
- Tables were the biggest loss-making product.
- Higher discounts negatively impacted profitability.

- ## Key Results

- Total Sales: $2.29M
- Total Profit: $286K
- Total Quantity Sold: 37,873
- Highest Sales Category: Technology
- Highest Profit Category: Technology
- Top State by Sales: California
- Biggest Loss-Making Product: Tables
